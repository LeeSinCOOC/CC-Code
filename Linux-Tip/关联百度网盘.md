### 1.安装库
```
pip install requests -i https://pypi.doubanio.com/simple
pip install bypy -i https://pypi.doubanio.com/simple
```

### 2.授权
```
bypy info
```
+ 得到一个链接，复制到浏览器打开
+ 复制此授权码，粘贴到linux命令行

### 3.数据放在我的应用数据“bypy”文件夹


### 4.下载
```
bypy downdir -v
bypy downdir dirname -v
```

### 5.上传
```
bypy upload -v
```

### 6.显示bypy列表文件
```
bypy list
```
